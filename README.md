## Overview

Native Instruments Play Series Hypr is built for production setups that call for a hybrid instrument with deep tonal shaping and animated movement. By combining wavetable and granular synthesis concepts with stacked sound design layers, it works well for evolving atmospheres, glitch-heavy patterns, and preset-based modern performances.

This release is tuned for a fast, responsive plugin workflow for users who want quick preset browsing, strong playability, and expanded modulation control in a single package. It includes MIDI 2.0 and MPE support, a macro modulation matrix, and live waveform visualization to give more immediate feedback while crafting sounds.
